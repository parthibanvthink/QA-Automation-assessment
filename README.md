# QA Automation Assessment – Web & Mobile

This QA Automation assessment covers both **Web** and **Mobile** platforms.

Please use the following applications for automation:

* **Web Application:** [https://www.saucedemo.com/](https://www.saucedemo.com/)
* **Mobile Application:** Pick any APK from the Play Store for mobile automation.

You may use **any programming language** or automation framework of your choice to complete the assessment.

## Assessment Requirements

The following areas will be evaluated:

### Step 1: GitHub Workflow (5 minutes)

* Clone the repository: https://github.com/parthibanvthink/QA-Automation-assessment.git
* Pull latest changes
* Create a folder with your name inside the repository, then work on your task within that folder.
  * Example: `Parthiban`
* Once complete, commit your changes and push them.

### Step 2: Test Plan (20 minutes)

Prepare a clear and structured test plan covering:

* Scope
* Test scenarios
* Approach
* Tools used
* Assumptions

### Step 3: Test Automation Script (25 minutes)

Develop automation test scripts with focus on:

* Coding standards
* Code reusability
* Proper project structure
* Maintainable design
* Clean and readable code

### Step 4: Test Report (10 minutes)

Generate an execution report that includes:

* Passed / Failed test cases
* Logs
* Screenshots (if applicable)
* Summary of execution results

## Evaluation Criteria

Your submission will be reviewed based on:

* Technical implementation
* Framework design
* Code quality
* Best practices
* Reporting quality
* Version control usage
